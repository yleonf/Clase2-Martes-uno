HTML y Python

1. HTML

HTML (Hypertext Markup Language) es un lenguaje de programación que se utiliza para crear y estructurar páginas web. Es el código que define la estructura de una página web y cómo se debe mostrar a los usuarios.

2. Python

Python es un lenguaje de programación de alto nivel, interpretado y de código abierto, conocido por su sintaxis clara y legible, que se asemeja al lenguaje natural, lo que facilita su aprendizaje y uso
